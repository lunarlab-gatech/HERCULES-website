# HERCULES Project Page

Source for the [HERCULES](https://github.com/lunarlab-gatech/HERCULES) project page: an open-source Unreal Engine 5 simulator for heterogeneous UAV–UGV teams supporting collaborative SLAM, cooperative perception, and large-scale exploration.

## Preview locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
